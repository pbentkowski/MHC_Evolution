24 How to compile:
25 -----------
26 The program was written in https://en.wikipedia.org/wiki/C%2B%2B14 C++14 standard so if you are using GCC then version gcc 4.8 seems to be the minimum requirement. This program has
27 some serious dependencies on http://www.boost.org/ C++ Boost Libraries. Should compile smoothly on most modern GNU/Linux distros with Boost Libs installed. Having http://www.scons.org/ Scons build tool might be useful too. Basic compilation works fine on Ubuntu 14.04 LTS as well as 16.04 LTS with mentioned packages installed by running the command:
28 
29 $** g++ -O3 -o MHC_model main.cpp Gene.cpp Antigen.cpp Host.cpp Pathogen.cpp H2Pinteraction.cpp RandomNumbs.cpp Tagging_system.cpp Environment.cpp DataHandler.cpp -std=c++14**
30 
31 It should also compile with flag -std=c++11 only with minor warnings. The code here can be also used like a toolbox for your research. You can stitch your own *main_yourown.cpp* file with your scenario and custom procedures (we did so for our research) and compile it using Scons script which is part of this code bundle. To do so run:
32 
33 $** scons -Q scenario="Scenarios/main_default.cpp"**
34 
35 replacing "Scenarios/main_default.cpp" with the path to your own custom *main_xyz.cpp*. Use our *main_core.cpp* file as the base to develop your own scenarios of MHC evolution dynamics. WARNING!
36 Using our Scons script overwrites the *main.cpp* file in the main source directory - always
37 store your scenarios somewhere else then the *main.cpp* file!
38 
39 Sometimes the HPC Cluster you have is lame and old and it has fairly outdated compiler (e.g. gcc < 4.8). Then you can statically link the libraries on your fancy brand new PC running "The Awesome Linux 3000" distro and send the no-dependencies executable to cluster. Compile like this:
40 
41 $ **g++ -static -O3 -o MHC_model main.cpp Gene.cpp Antigen.cpp Host.cpp Pathogen.cpp H2Pinteraction.cpp RandomNumbs.cpp Tagging_system.cpp Environment.cpp DataHandler.cpp -std=c++14**
42 
43 Or run the Scons script:
44 
45 $** scons -Q scenario="Scenarios/main_default.cpp" linking="static" **
