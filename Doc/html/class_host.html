chromovector Host::doCrossAndMeiosis ( double  corssing_prob)

Core method. Performing crossing over and chromosome selection before mating.

It mixes up genes from two host chromosomes to form one child chromosome later pushed for mating. One can regulate which chromosome is preferred as a source of genes by changing crossing-over probability parameter.

Parameters
corssing_prob-probability of a gene crossing over; p=0.5 means that each chromosome gives half of its genes to the resulting meiotic chromosome. p lesser then 0.5 means ChromosomeOne will give majority of genes; p larger then 0.5 will favour ChromosomeTwo as a donor.
Returns
a Chromosome vector.
