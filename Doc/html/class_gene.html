Core method. Iterates through a gene sequence and (if selected so) flips the value of a single bit to an opposite one. But certain bits are not permitted to mutate - these are specified in a global std::set.

Parameters
mut_probabl- probability of mutating a single bit.
timeStamp- current time (current number of the model iteration).
noMutts- a std::set containing indices of residues of the bit-string that are not allowed to change.
