void Pathogen::setNewPathogen ( int  num_of_loci,
int  gene_size,
int  species,
int  timeStamp 
)

Core method. Sets a new Pathogen object.

Sets a new Pathogen object and assigns a user-defined number of genes of user-defined number of bits (as genes are bit-strings). Value of a gene is randomly assigned from an interval between 0 and 2^N-1, where N is the number of bits in a gene.

Parameters
num_of_loci- number of gene loci in a chromosome
gene_size- the length of the bit-string representing a gene
species- user-defined number of species
timeStamp- current time (current number of the model iteration)

Core method. Sets a new Pathogen object.

Sets a new Pathogen object and assigns a user-defined number of genes of user-defined number of bits (as genes are bit-strings). Value of a gene is randomly assigned from a user-specified interval.

Parameters
num_of_loci- number of gene loci in a chromosome
gene_size- the length of the bit-string representing a gene
species- user-defined number of species
low_lim- lower limit of the range of permitted bit-string values
up_lim- upper limit of the range of permitted bit-string values
timeStamp- current time (current number of the model iteration)
